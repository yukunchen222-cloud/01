# AI短剧推广自动剪辑工作流

## 项目简介

这是一个基于LangGraph构建的AI短剧推广自动剪辑工作流系统。系统能够：
1. 自动分析用户上传的短剧素材
2. 搜索并学习同类爆款视频的成功要素
3. 生成详细的剪辑策略文档
4. 审核剪辑成品质量

## 核心功能

### 🎬 素材智能分析
- 提取视频关键帧
- 生成视频字幕
- 识别剧情类型和情感走向
- 提取搜索关键词

### 🔥 爆款学习
- 搜索抖音/快手同类爆款视频
- 分析钩子设计模式
- 解构标题技巧
- 研究封面策略
- 总结可复制的爆款公式

### 📝 策略生成
- 基于素材与爆款的匹配度分析
- 智能选择最佳钩子类型
- 规划剪辑节奏和镜头选择
- 生成标题和封面建议
- 制定音频策略

### ✅ 质量审核
- 评估钩子执行效果
- 检查剪辑节奏
- 对比爆款特征
- 给出改进建议

## 快速开始

### 安装依赖

```bash
# 使用 uv 安装依赖
uv sync
```

### 运行工作流

```python
from graphs.graph import main_graph
from graphs.state import GraphInput
from utils.file.file import File

# 准备输入
input_data = GraphInput(
    material_video=File(url="https://example.com/video.mp4"),
    drama_type="都市情感"
)

# 执行工作流
result = main_graph.invoke(input_data)

# 查看剪辑策略
import json
print(json.dumps(result.get("edit_strategy", {}), indent=2, ensure_ascii=False))
```

## 配置说明

### 大模型配置

配置文件位于 `config/` 目录：

- `material_analyze_llm_cfg.json`: 素材分析配置
- `viral_analyze_llm_cfg.json`: 爆款分析配置
- `edit_strategy_llm_cfg.json`: 策略生成配置
- `review_llm_cfg.json`: 审核比对配置

### 素材库配置

- **输入素材库**: 放置待剪辑的原始素材视频
- **输出素材库**: 存放剪辑完成的成品视频

## 技术栈

- **框架**: LangGraph 1.0
- **语言**: Python 3.11+
- **大模型**: DeepSeek, 豆包 (多模态)
- **视频处理**: coze-coding-dev-sdk
- **Web搜索**: coze-coding-dev-sdk

## 注意事项

1. 工作流执行需要网络连接（用于搜索爆款视频）
2. 视频URL需要可公开访问
3. 策略生成后需要人工确认
4. 大模型调用会产生Token消耗
